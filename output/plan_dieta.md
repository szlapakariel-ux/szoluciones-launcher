# Plan de trabajo — idea de dieta

## Objetivo
Construir un sistema simple para llevar el seguimiento de una dieta personal, con registro de alimentos, metas y recordatorios leves.

## Pasos
1. Confirmar el alcance mínimo: registro de comidas, metas calóricas o de macros y alertas básicas.
2. Definir los datos principales: usuario, alimentos, comidas, metas y recordatorios.
3. Implementar el flujo de alta/edición de registros y la lógica de notificaciones.
4. Preparar validaciones de entrada y pruebas básicas de humo.

## Criterios de salida
- El usuario puede registrar comidas y revisar el estado diario.
- El sistema tiene una regla mínima para evitar mostrar datos sin contexto válido.
- El desarrollo queda documentado con referencias a los tornillos de ADN.
